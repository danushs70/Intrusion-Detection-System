## Dataset

This project uses the **CICIDS-2017** dataset developed by the **Canadian Institute for Cybersecurity (CIC), University of New Brunswick (UNB)**.

Due to GitHub's file size limitations, the dataset is **not included** in this repository.

### Download the dataset

- Official CICIDS-2017 Dataset: https://www.unb.ca/cic/datasets/ids-2017.html
- Kaggle Mirror (CSV format): https://www.kaggle.com/datasets/dhoogla/cicids2017

After downloading, place the dataset in the following directory:

```text
dataset/
└── cicids.csv
```
